Week 1: Arrays, Functions, and Objects (Basic Foundation)
// Basic Access
ได้เรียนรู้ในเรื่องของ การใช้ Array ในการเก็บข้อมูล และฝึกการใช้คำสั่ง push(ในการเพิ่มข้อมูล) และ pop(ในการลบข้อมูล)

// Loop & Calculation
ได้เรียนรู้ในเรื่องของการ วนซ้ำและนำข้อมูลทั้งหมดใน Array มาใช้ในการคำนวณ เพื่อหาผลรวม

// Array Manipulation
ได้เรียนรู้การจัดการข้อมูลใน Array การรับค่า และเพิ่มค่าให้กับข้อมูลก่อนส่งกลับออกมา

//forEach
ได้รู้จัก method ที่สามารถใช้ในการวนซ้ำ ข้อมูลใน Array โดยไม่ต้องเขียน Function เองโดยการใช้ Forloop
